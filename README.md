A simple Single Page Application task setter built with Laravel/Vuejs. Designed to be simple, efficient and easily scalable. Supports API authentication with Laravel Passport
 
# Technology Stack

* PHP/Laravel
* JavaScript/Vue Js
* Twitter Bootstrap
* Animate CSS
